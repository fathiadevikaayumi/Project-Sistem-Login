# Project-Sistem-Login
Aplikasi sistem login sederhana berbasis web menggunakan HTML dan PHP. Pengguna dapat memasukkan username dan password untuk mengakses halaman sistem. Project ini dibuat sebagai latihan memahami autentikasi pengguna dan pengolahan form pada aplikasi web.

## Fitur
* Form login menggunakan username dan password
* Validasi data login
* Halaman berhasil login
* Pesan kesalahan jika login gagal

## Cara Menggunakan
1. Jalankan server lokal seperti XAMPP atau Laragon.
2. Simpan project di folder **htdocs**.
3. Buka browser dan akses project melalui localhost.
4. Masukkan username dan password pada form login.

## Teknologi yang Digunakan
* HTML
* PHP

## Tujuan Project
Project ini dibuat untuk mempelajari:
* Dasar pembuatan form login
* Pengolahan data menggunakan PHP
* Konsep autentikasi sederhana pada aplikasi web
